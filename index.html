Threads Social Network Analysis (Java)

Overview

This project models Threads as a directed graph using Java.

Data Structures Used

  • HashMap (Adjacency List)
  • ArrayList
  • Queue (BFS)
  • HashSet

Algorithms Implemented

  • Breadth-First Search (Shortest Path)
  • Degree Centrality

Sample Output

THREADS SOCIAL NETWORK ANALYSIS

Bob -> [David]
Carol -> [Bob]
David -> []
